# Security Policy

## Supported Versions

Only the latest released version of this action receives security updates.

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it privately so it can be addressed before public disclosure.

- Preferred: open a [private security advisory](https://github.com/antoniovazquezblanco/setup-ghidra/security/advisories/new) through GitHub.
- Alternatively, open a regular [issue](https://github.com/antoniovazquezblanco/setup-ghidra/issues) for low-severity reports.

Please include:

- A description of the vulnerability and its impact.
- Steps to reproduce or a proof of concept.
- Any suggested remediation, if available.

You can expect an initial response within a reasonable timeframe. Once a fix is available, a new release will be published and the advisory disclosed.
